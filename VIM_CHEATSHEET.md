# Vim 速查卡

> 基于本 dotsfile 的 `.vimrc`。带 ★ 的是本配置自定义的快捷。Leader = `,`

---

## 模式

| 进入                    | 模式                       |
| ----------------------- | -------------------------- |
| `Esc`                   | Normal（老家）             |
| `i` `a` `I` `A` `o` `O` | Insert                     |
| `v` `V` `Ctrl-v`        | Visual（字符 / 行 / 列块） |
| `:`                     | Command                    |

---

## 移动

| 键                    | 作用                   |
| --------------------- | ---------------------- |
| `h j k l`             | ← ↓ ↑ →                |
| `w / b / e`           | 词首前 / 词首后 / 词尾 |
| `0 / ^ / $`           | 行首 / 首非空 / 行尾   |
| `f<x>` `F<x>` `;` `,` | 行内跳到字符           |
| `gg / G / 12G`        | 文件首 / 尾 / 第 12 行 |
| `%`                   | 跳匹配括号             |
| `H / M / L`           | 屏幕顶 / 中 / 底       |
| `Ctrl-d / Ctrl-u`     | 半屏下 / 上            |
| `Ctrl-f / Ctrl-b`     | 全屏下 / 上            |
| `Ctrl-o / Ctrl-i`     | 跳转历史 退/进         |
| 重开文件 ★            | 自动回到上次光标位置   |

---

## 编辑（动词 + 范围）

| 动词      | 含义                |
| --------- | ------------------- |
| `d`       | 删除 / 剪切         |
| `c`       | 改（删 + 进入插入） |
| `y`       | 复制                |
| `>` `<`   | 缩进                |
| `gu` `gU` | 转小/大写           |

| 范围                          | 含义                                  |
| ----------------------------- | ------------------------------------- |
| `w` `b` `e`                   | 一个词                                |
| `$` `0`                       | 到行尾 / 行首                         |
| `iw` / `aw`                   | 内 / 含外侧空白的词                   |
| `i"` `i'` `i(` `i[` `i{` `it` | 包裹内（双引号 / 单引 / 括号 / 标签） |
| `a"` `a(` ...                 | 含包裹符本身                          |
| `gg` `G`                      | 整文件                                |

例：`ci"` 改双引号内 / `dap` 删一段 / `yi(` 复制括号内 / `>ip` 整段右移

| 速记        | 同义                                     |
| ----------- | ---------------------------------------- |
| `dd`        | 删整行                                   |
| `yy`        | 复制整行                                 |
| `cc`        | 改整行                                   |
| `D / C / Y` | 等价 `d$ / c$ / y$`（Y 在某些版本同 yy） |
| `x / X`     | 删光标处 / 前                            |
| `r<x>`      | 替换光标字符                             |
| `~`         | 切换大小写                               |

---

## 撤销 / 重做 / 重复

| 键            | 作用                          |
| ------------- | ----------------------------- |
| `u`           | 撤销                          |
| `Ctrl-r`      | 重做                          |
| `.`           | 重复上次修改（杀器）          |
| 关 vim 再开 ★ | `u` 仍能撤销（持久 undofile） |

## 搜索

| 键                    | 作用                |
| --------------------- | ------------------- |
| `/pat` `?pat`         | 向后 / 向前搜       |
| `n / N`               | 下一 / 上一命中     |
| `*  #`                | 搜光标下词（前/后） |
| `Esc Esc` ★ 或 `,h` ★ | 清搜索高亮          |

---

## 替换

| 命令           | 范围              |
| -------------- | ----------------- |
| `:s/a/b/`      | 当前行第一个      |
| `:s/a/b/g`     | 当前行全部        |
| `:%s/a/b/g`    | 全文件            |
| `:%s/a/b/gc`   | 全文件 + 逐个确认 |
| `:5,15s/a/b/g` | 第 5-15 行        |

---

## 项目级查找（无插件） ★

| 命令               | 作用                              |
| ------------------ | --------------------------------- |
| `:find foo<Tab>`   | 全项目模糊找文件名（`path=.,**`） |
| `,f` ★             | 等价 `:find `                     |
| `:grep pat`        | 项目内搜内容（自动用 ripgrep）    |
| `,g` ★             | 等价 `:grep `                     |
| `:copen` `:cclose` | 打开 / 关闭结果列表               |
| `:cnext` `:cprev`  | 跳下一个 / 上一个命中             |

---

## 文件 / Buffer / Window

### 保存退出

| 键          | 作用       |
| ----------- | ---------- |
| `:w` `,w` ★ | 保存       |
| `:q` `,q` ★ | 退出       |
| `:wq` `:x`  | 保存并退出 |
| `:q!`       | 强制退出   |

### Buffer（同时开多个文件）

| 键                | 作用               |
| ----------------- | ------------------ |
| `:e file`         | 打开文件           |
| `:ls` `,b` ★      | 列出所有 buffer    |
| `:b 3` / `:b foo` | 跳到 buffer        |
| `,n / ,p` ★       | 下一 / 上一 buffer |
| `,d` ★            | 关 buffer          |

### Window 分屏

| 键                    | 作用              |
| --------------------- | ----------------- |
| `:sp` `:vsp`          | 横 / 竖分屏       |
| `Ctrl-w h/j/k/l`      | 在 window 间移动  |
| `Ctrl-w c` `Ctrl-w o` | 关当前 / 只留当前 |
| `Ctrl-w =`            | 平均大小          |
| `Ctrl-w + - > <`      | 调整高度 / 宽度   |

### 文件树（netrw 内置）

| 键            | 作用                 |
| ------------- | -------------------- |
| `:Lex` `,e` ★ | 左侧打开/关闭文件树  |
| 树内 `Enter`  | 进入 / 打开          |
| 树内 `-`      | 上级目录             |
| 树内 `d D R`  | 新建 / 删除 / 重命名 |

## 可视模式

| 进入     | 模式                 |
| -------- | -------------------- |
| `v`      | 字符                 |
| `V`      | 整行                 |
| `Ctrl-v` | 列块（批量前缀杀器） |

选中后：`d / y / c / > / <`、`r<x>` 替换全部
列块技巧：`Ctrl-v` 选好 → `I前缀<Esc>` 给每行同时加前缀

## 寄存器 / 剪贴板 ★

| 键          | 作用                                      |
| ----------- | ----------------------------------------- |
| `"ay` `"ap` | 用寄存器 a 复制 / 粘贴                    |
| `"+y` `"+p` | 显式系统剪贴板                            |
| `:reg`      | 查看所有寄存器                            |
| 默认 ★      | `clipboard=unnamed`，y/p 自动通系统剪贴板 |

## 折叠 ★

| 键          | 作用                  |
| ----------- | --------------------- |
| `<Space>` ★ | 当前块开/合（自定义） |
| `zo / zc`   | 展开 / 折叠           |
| `zR / zM`   | 全展开 / 全折         |

---

## 宏（录制按键）

1. `qa` 开始录到寄存器 a
2. 做操作
3. `q` 停止
4. `@a` 回放；`@@` 重复上次；`100@a` 执行 100 次

---

## 命令行小技巧

| 键             | 作用                     |
| -------------- | ------------------------ |
| `:` 后 `Tab` ★ | 可视补全菜单（wildmenu） |
| `q:`           | 命令历史窗（Enter 重发） |
| `q/`           | 搜索历史窗               |
| `:!cmd`        | 跑 shell 命令            |
| `:r !cmd`      | 把命令输出插到光标处     |

---

## Marks 书签

| 键       | 作用                  |

| `` `a `` | 跳到书签 a 的精确位置 |
| `'a`     | 跳到书签 a 所在行行首 |
